Настроить jump хост для секурного доступа к приватной сетке

1. SSH конфигурация:
   - Отрубить root 
   - Поменять порт на 2222
   - Разрешить только группу `ssh-users`
   
2. Настроить SSH туннелирование:
   - Разрешить port forwarding
   - Запретить X11 forwarding
   - Настроить idle timeout 10 минут
   
3. Логирование и аудит:
   - Логировать все команды (auditd или аналог)
   - Отправлять алерты о новых подключениях
   
4. Сетевые правила (iptables):
   - Разрешить SSH только с определенных IP
   - Разрешить форвард портов только в приватную сеть
   - Блокировать исходящие соединения в интернет
   - Rate limiting на SSH
   
5. Настроить 2FA (Google Authenticator PAM)

6. Создать хелп скрипт для пользователей:
   - Автоматическое создание SSH конфига
   - Примеры команд для проброса портов
